# Desenvolvimento Web Responsivo - UNIP 2026

Repositório criado para armazenar conteúdos, exercícios e projetos desenvolvidos durante as aulas de Desenvolvimento Web Responsivo.

## Objetivo

Este repositório tem como finalidade:

- Praticar HTML, CSS e JavaScript
- Desenvolver layouts responsivos
- Aplicar boas práticas de desenvolvimento web
- Organizar os projetos e atividades realizadas em aula

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura

Os arquivos estão organizados por aulas, exemplos e exercícios práticos.
